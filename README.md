# ProsCons株式会社　提出課題

### python開発環境
Python 3.8.5

### リポジトリの作成
`$ git clone https://github.com/Shun712/pros_cons_test.git`
### 作業ディレクトリを作成
`$ mkdir test`
### 作業ディレクトリに移動
`$ cd pros_cons_test`
### venvで仮想環境を作成
`$ python3 -m venv testenv`
### 仮想環境に入る
`$ source testenv/bin/activate`
### テストファイル実行
`$ python test.py`
